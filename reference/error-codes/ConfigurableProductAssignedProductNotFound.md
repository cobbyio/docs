
# ConfigurableProductAssignedProductNotFound

### Message
Configurable product: The assigned product SKU does not exist.

### Description
If you assign a SKU to a configurable product in cobby which does not exist (e.g. because it has been deleted in Magento) you will get the following error:

![](/img/import-error-codes/pic-21.png)

### Solution
To fix this error, remove the affected SKU from the mapping.
