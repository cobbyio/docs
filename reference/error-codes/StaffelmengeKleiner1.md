
# StaffelmengeKleiner1/TierPriceLessThan1

### Message
Tier price: quantity is smaller than 1.

### Description
If the entered quantity is less than 1, the field and the number turn red. The marked values are also not taken over. 

![](/img/import-error-codes/pic-32.png)

### Solution
Change the quantity to anything greater than 0.
