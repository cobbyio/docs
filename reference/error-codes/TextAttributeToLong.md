
# TextAttributeToLong

### Message
The text attribute is too long.

### Description
There are attributes like "Description" which ideally should not be longer than 255 characters. If they are longer than 255 characters, you will get a message like this:

![](/img/import-error-codes/pic-1.png)

### Solution
This message is created in cobby so that the user can refer to the essential information of a product and thus improve their SEO (search engine optimization). You get around this problem by shortening the text to under 255 characters.
