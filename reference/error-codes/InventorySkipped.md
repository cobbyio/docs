
# InventorySkipped

### Message 
Change is not possible, stock management disabled

### Description
The affected cobby user is not authorized to make changes in Excel.

### Solution
The stock management in Magento has to be adjusted because it is set to read-only/disabled. Change the value "Stock Settings" in Magento.
