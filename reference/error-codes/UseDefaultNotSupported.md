
# UseDefaultNotSupported

### Message
Use of “Set Default Value”/cobby default in wrong field.

### Description
If you use the function "Set Default Value" in a wrong field (e.g. Images), this leads to problems.

![](/img/import-error-codes/pic-19.png)

You will get the following error message after saving:

![](/img/import-error-codes/pic-20.png)

### Solution
Apply this function only in valid application areas to avoid this error.
