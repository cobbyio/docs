
# Neues Produkt zu validierende Spalte ausgeblendet

### Message
New product: column to be validated is hidden.

### Description
If a new product was created and columns were hidden that are mandatory fields, the following error message appears:

![](/img/import-error-codes/pic-13.png)

### Solution
To fix this error, display the column (at least while creating the product) and enter a valid value.
