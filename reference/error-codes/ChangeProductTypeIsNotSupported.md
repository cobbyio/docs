
# ChangeProductTypeIsNotSupported

### Message
Change of product type is not supported.

### Description
If you change the product type of an item, you will receive the following error message in the Error Task Pane:

![](/img/import-error-codes/pic-8.png)

### Solution
Magento in version 1 does not allow changing the type. To work around this error, do not change this attribute and leave the previous settings.
