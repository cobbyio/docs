
# ConfigurableProductAssignedAttributeSetTypeNotSupported

### Message
Configurable assignment: Wrong attribute set has been assigned.

### Description
If you assign an item from a different attribute set to a configurable product, you receive the following error message:

![](/img/import-error-codes/pic-23.png)

### Solution
To fix this error, select products that are in the attribute set. Alternatively, you can create new products in the attribute set and link them afterwards.
