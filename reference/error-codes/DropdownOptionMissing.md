
# DropdownOptionMissing

### Message
Invalid selection in multiselect fields: The drop-down option is missing.

### Description
As in Magento, there are also attributes in cobby whose content is predefined as a drop-down menu. This means that no individual new values can be entered there. If you try to enter an invalid value, the following message will appear:

![](/img/import-error-codes/pic-2.png)

### Solution
To correct this error, select a valid value from the drop-down menu.
