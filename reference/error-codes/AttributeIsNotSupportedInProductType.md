
# AttributeIsNotSupportedInProductType

### Message 
The attribute is not supported in product type.

### Description
This error occurs if an attribute is only assigned to the product types simple and configurable and it is still tried to set another type. 

![](/img/import-error-codes/pic-37.png)

### Solution
Fix this by changing the attribute settings in the Magento backend, e.g. navigate to color and set the checkmark for the desired product types.

![](/img/import-error-codes/pic-38.png)
