---
title: Price 0 (Error Code 51) - Note Option
date: 2024-01-17
---

Due to safety reasons, error code 51 informs you that the price of some SKUs is 0.

It is now possible to always ignore this note or to ignore it only once by clicking on the three dots in the error assistant task pane.

![](/img/release-notes/price0-errorcode51.png)

<!-- truncate -->

_Note: If error code 51 doesn't appear at all and the green exclamation mark icon (below) is shown, please check your Excel version and update it to version 2.4.1.12 or 2.4.1.20._

![](/img/release-notes/note.PNG)
